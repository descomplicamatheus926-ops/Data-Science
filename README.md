# Data-Science
NotebookLM com as algumas fontes para otimizar o aprendizado.

Essas fontes educacionais oferecem uma visão abrangente sobre o ecossistema de análise de dados e a infraestrutura de Big Data. 
Os materiais explicam como os sistemas de informação e a ciência de dados apoiam a tomada de decisão estratégica através do uso de ferramentas como Power BI, Hadoop e bancos de dados NoSQL, SQL entre outros.
